# ระบบจับฉลากแบ่งสายฟุตบอล 7 คน วันรพี 69 — v1.4

## ใช้งานวันงาน

1. เปิด `เปิดระบบ_Windows.bat` แล้วรอให้เบราว์เซอร์เปิดเอง
2. เปิดหน้าควบคุม และกด **เปิดหน้าจอนำเสนอ**
3. ดูรายการ **ตรวจความพร้อมก่อนเริ่ม** ให้ครบทั้ง 4 ข้อก่อนเริ่มถ่ายทอดสด
4. แชร์เฉพาะหน้าจอนำเสนอใน Google Meet

หน้าควบคุมจะแสดงสถานะ “จอนำเสนอเชื่อมต่อแล้ว” ต่อเมื่อได้รับสัญญาณจากหน้าจอนำเสนอจริงภายใน 5 วินาที

## การซ้อมและถ่ายทอดสด

การสลับโหมดจะล้างผลปัจจุบันทุกครั้งโดยตั้งใจ เพื่อไม่ให้ผลซ้อมติดออกอากาศ โหมดถ่ายทอดสดจึงเริ่มจากข้อมูลว่างเสมอ

การล็อกผลเป็นเพียงตัวกันกดพลาด ไม่ใช่ระบบรักษาความปลอดภัย การปลดล็อกต้องพิมพ์คำว่า `ปลดล็อก` และไม่มีรหัสลับอยู่ในไฟล์สาธารณะ

## ซิงก์คนละเครื่อง (Google Apps Script)

ค่าเริ่มต้นซิงก์ได้เฉพาะแท็บ/เบราว์เซอร์ในเครื่องเดียวกัน หากหน้าควบคุมและหน้าจอนำเสนออยู่คนละเครื่อง ให้ทำครั้งเดียวก่อนงาน:

1. สร้างโปรเจ็กต์ที่ [Google Apps Script](https://script.google.com/) แล้ววางเนื้อหาจาก `GAS_BACKEND.gs`
2. Deploy เป็น **Web app** โดยอนุญาตให้ผู้ใช้งานตามที่หน่วยงานกำหนด (ต้องให้เครื่องหน้าควบคุมและจอนำเสนอเรียก URL นี้ได้)
3. คัดลอก URL ที่ลงท้ายด้วย `/exec` ไปใส่ใน `sync-config.js`
4. อัปโหลดไฟล์ทั้งหมดไปยัง GitHub Pages ใหม่ แล้วเปิดทั้งสองเครื่องจาก URL เดียวกัน

บริการนี้เก็บเฉพาะสถานะการจับฉลากและสัญญาณว่าจอนำเสนอยังเปิดอยู่ จึงควรจำกัดสิทธิ์การเข้าถึงของ Web app ให้เหมาะกับงาน

## ส่งออกผล

- **พิมพ์ / บันทึก PDF**: เปิดหน้าสรุปโดยเฉพาะ ไม่ขึ้นกับหน้าที่กำลังออกอากาศ
- **บันทึกสรุปเป็น PNG**: ดาวน์โหลดภาพสรุปสำหรับส่งในกลุ่มทันที (ต้องเชื่อมอินเทอร์เน็ตเพื่อโหลดตัวช่วยครั้งแรก)
- **ดาวน์โหลดไฟล์สำรอง JSON**: เก็บสำรองสถานะดิบ

บนหน้าจอนำเสนอกด `1`–`5` เพื่อเปลี่ยนหน้า และกด `F` เพื่อเต็มจอ
